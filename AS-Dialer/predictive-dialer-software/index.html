    WHAT IS A PREDICTIVE DIALER?

    A Predictive dialer uses an algorithm based on agent performance metrics and ensures that the agents get maximum potential calls. It is a smart outbound dialer that autonomously dials multiple numbers simultaneously from a provided contact list, and only assigns live calls to the agents. Such a dialer proves to be an essential tool for utilizing idle time of the agents, thus ensuring more calls to sales conversions.
